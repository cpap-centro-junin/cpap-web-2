# 1. INTRODUCCIÓN

## 1.1. Propósito del Manual

Este manual tiene como objetivo guiar a los usuarios administrativos del sistema web institucional del Colegio Profesional de Antropólogos del Perú - Región Centro (CPAP) en el uso correcto y eficiente del panel administrativo.

El manual está diseñado para proporcionar instrucciones claras y detalladas sobre todas las funcionalidades disponibles en el sistema, facilitando la gestión de contenidos, colegiados, documentos y demás elementos del sitio web institucional.

**A quién está dirigido:**
- Administradores del CPAP Región Centro
- Personal autorizado para la gestión del sitio web
- Miembros del consejo directivo con acceso al sistema

## 1.2. Alcance del Sistema

El sistema web institucional del CPAP - Región Centro es una plataforma completa desarrollada en Laravel que permite:

✅ **Gestión de Contenidos:**
- Noticias institucionales
- Eventos y actividades
- Galería fotográfica
- Biblioteca virtual

✅ **Gestión de Colegiados:**
- Registro de antropólogos colegiados
- Emisión de documentos de habilitación con código QR
- Control de visibilidad de información personal
- Verificación pública de habilitaciones

✅ **Servicios:**
- Bolsa de trabajo
- Mensajes de contacto
- Documentos normativos

✅ **Personalización:**
- Configuración de página de inicio
- Editor visual de colores del sitio
- Gestión del consejo directivo

## 1.3. Usuarios del Sistema

El sistema maneja diferentes niveles de acceso:

**Administrador Principal:**
- Acceso completo a todas las funcionalidades
- Gestión de usuarios e invitaciones
- Configuración del sistema

**Administrador de Contenidos:**
- Gestión de noticias, eventos y galería
- Respuesta a mensajes de contacto
- Actualización de información institucional

**Administrador de Colegiados:**
- Registro y edición de colegiados
- Emisión y revocación de habilitaciones
- Gestión del directorio profesional

> **Nota:** Los permisos específicos de cada usuario son configurados por el administrador principal.

## 1.4. Convenciones del Manual

A lo largo de este manual se utilizarán los siguientes elementos visuales:

> 💡 **Consejo:** Recomendaciones y mejores prácticas

> ⚠️ **Advertencia:** Información importante que debe tenerse en cuenta

> ⛔ **Cuidado:** Acciones que pueden tener consecuencias irreversibles

> 📝 **Nota:** Información adicional relevante

> ✅ **Ejemplo:** Casos prácticos de uso

**Elementos de la interfaz:**
- Los nombres de botones aparecen en **negrita**
- Las rutas del menú se muestran como: Menú > Submenu > Opción
- Los campos de formularios aparecen en *cursiva*

## 1.5. Requisitos del Sistema

### Requisitos Mínimos del Navegador:

**Navegadores Compatibles:**
- Google Chrome 90 o superior ✅ (Recomendado)
- Mozilla Firefox 88 o superior ✅
- Microsoft Edge 90 o superior ✅
- Safari 14 o superior ✅

**Resolución de Pantalla:**
- Mínima: 1280 x 720 píxeles
- Recomendada: 1920 x 1080 píxeles

**Conexión a Internet:**
- Velocidad mínima: 2 Mbps
- Recomendada: 5 Mbps o superior

**Otros Requisitos:**
- JavaScript habilitado
- Cookies habilitadas
- Lector de PDF (para visualización de documentos)

> 💡 **Consejo:** Se recomienda utilizar Google Chrome para una mejor experiencia debido a su compatibilidad completa con todas las funcionalidades del sistema.

# 2. ACCESO AL SISTEMA

## 2.1. Acceso a la Plataforma

Para acceder al panel administrativo del sistema CPAP, debe contar con credenciales de acceso proporcionadas por el administrador principal.

**URL de Acceso:**
```
https://www.cpapregioncentro.com/login
```

> 📝 **Nota:** Reemplace la URL con el dominio real del sistema una vez esté en producción.

## 2.2. Inicio de Sesión

**Paso 1:** Abra su navegador web preferido (se recomienda Google Chrome)

**Paso 2:** Ingrese la URL del sistema en la barra de direcciones

**Paso 3:** Se mostrará la pantalla de inicio de sesión con los siguientes elementos:
- Logotipo institucional del CPAP
- Formulario de inicio de sesión
- Enlace "¿Olvidaste tu contraseña?"

**[🖼️ IMAGEN 2.2: Captura de pantalla completa de la página de login, mostrando el formulario con los campos de email y contraseña, y el botón "Iniciar Sesión"]**

**Paso 4:** Complete el formulario con sus credenciales:

- **Campo *Correo Electrónico*:** Ingrese su dirección de correo electrónico registrada
- **Campo *Contraseña*:** Ingrese su contraseña

**[🖼️ IMAGEN 2.3: Primer plano del formulario de login con los campos llenos (puede difuminar los datos reales)]**

**Paso 5:** Haga clic en el botón **"Iniciar Sesión"**

**Paso 6:** El sistema validará sus credenciales:

✅ **Si las credenciales son correctas:**
- Será redirigido automáticamente al Dashboard principal
- Verá un mensaje de bienvenida con su nombre

⛔ **Si las credenciales son incorrectas:**
- Aparecerá un mensaje de error en color rojo
- Los campos se limpiarán para que intente nuevamente

**[🖼️ IMAGEN 2.4: Captura de un mensaje de error al ingresar credenciales incorrectas]**

> ⚠️ **Advertencia:** Después de 5 intentos fallidos consecutivos, su cuenta será bloqueada temporalmente por 30 minutos por seguridad. Contacte al administrador si esto ocurre frecuentemente.

## 2.3. Recuperación de Contraseña

Si olvidó su contraseña, puede recuperarla siguiendo estos pasos:

**Paso 1:** En la pantalla de inicio de sesión, haga clic en el enlace **"¿Olvidaste tu contraseña?"**

**[🖼️ IMAGEN 2.5: Captura mostrando el enlace de recuperación de contraseña]**

**Paso 2:** Será redirigido a la página de recuperación de contraseña

**[🖼️ IMAGEN 2.6: Página completa de recuperación de contraseña]**

**Paso 3:** Ingrese su dirección de correo electrónico registrada en el campo *Correo Electrónico*

**Paso 4:** Haga clic en el botón **"Enviar Enlace de Recuperación"**

**Paso 5:** El sistema enviará un correo electrónico a la dirección proporcionada con un enlace para restablecer su contraseña

**[🖼️ IMAGEN 2.7: Mensaje de confirmación de envío de correo de recuperación]**

**Paso 6:** Revise su bandeja de entrada (y carpeta de spam/correo no deseado)

**Paso 7:** Abra el correo y haga clic en el enlace de recuperación

**[🖼️ IMAGEN 2.8: Ejemplo del correo electrónico de recuperación (puede ser captura o mockup)]**

**Paso 8:** Será redirigido a una página para establecer su nueva contraseña

**[🖼️ IMAGEN 2.9: Formulario de restablecimiento de contraseña]**

**Paso 9:** Ingrese su nueva contraseña dos veces:
- Campo *Nueva Contraseña*
- Campo *Confirmar Contraseña*

**Paso 10:** Haga clic en el botón **"Restablecer Contraseña"**

**Paso 11:** Su contraseña será actualizada y podrá iniciar sesión con la nueva contraseña

> 💡 **Consejo:** Elija una contraseña segura que contenga al menos 8 caracteres, incluyendo mayúsculas, minúsculas, números y símbolos.

> 📝 **Nota:** El enlace de recuperación expira después de 60 minutos por seguridad. Si no lo utiliza a tiempo, deberá solicitar uno nuevo.

## 2.4. Cerrar Sesión

Para cerrar sesión de forma segura:

**Paso 1:** Haga clic en su nombre de usuario ubicado en la esquina superior derecha del panel

**[🖼️ IMAGEN 2.10: Captura mostrando el menú de usuario desplegado en la esquina superior derecha]**

**Paso 2:** En el menú desplegable, haga clic en la opción **"Cerrar Sesión"**

**Paso 3:** Será redirigido automáticamente a la página de inicio de sesión

**Paso 4:** Verá un mensaje de confirmación indicando que ha cerrado sesión correctamente

> ⚠️ **Advertencia:** Siempre cierre sesión cuando termine de usar el sistema, especialmente si está usando una computadora compartida o pública.

## 2.5. Seguridad y Recomendaciones

### Recomendaciones de Seguridad:

✅ **Hacer:**
- Utilice contraseñas únicas y seguras
- Cambie su contraseña periódicamente (cada 90 días)
- Cierre sesión al terminar de usar el sistema
- Verifique que está en el sitio web oficial antes de ingresar credenciales
- Mantenga su correo electrónico actualizado

⛔ **No Hacer:**
- No comparta sus credenciales con otras personas
- No use la misma contraseña en múltiples sitios
- No deje su sesión abierta en computadoras públicas
- No anote sus contraseñas en lugares visibles
- No haga clic en enlaces sospechosos en correos electrónicos

### Características de una Contraseña Segura:

- Mínimo 8 caracteres
- Al menos una letra mayúscula (A-Z)
- Al menos una letra minúscula (a-z)
- Al menos un número (0-9)
- Al menos un símbolo especial (@, #, $, %, etc.)

✅ **Ejemplo de contraseña segura:** `Cpap2026!Centro`
⛔ **Ejemplo de contraseña débil:** `12345678` o `password`

> 💡 **Consejo:** Utilice un gestor de contraseñas como LastPass, 1Password o Bitwarden para almacenar sus credenciales de forma segura.

### Buenas Prácticas para Banner Slides

✅ **Recomendaciones:**

**Cantidad de Slides:**
- Mínimo: 3 slides para un carrusel dinámico
- Óptimo: 4-5 slides
- Máximo: 7 slides (más de esto puede ser abrumador)

**Diseño de Imágenes:**
- Use imágenes de alta calidad y resolución
- Dimensiones recomendadas: 1920 x 1080 píxeles (proporción 16:9)
- Mantenga los elementos importantes en el centro de la imagen
- Evite imágenes con mucho texto
- Use colores que contrasten con los colores institucionales

**Contenido de Texto:**
- Título: conciso y llamativo (máximo 10 palabras)
- Descripción: clara y breve (máximo 20 palabras)
- Tag: una sola palabra impactante

**Orden de los Slides:**
- Coloque el contenido más importante en el slide #1
- Ordene por prioridad o actualidad
- Considere rotarlos mensualmente para mantener el sitio fresco

**Botones de Acción:**
- Use verbos de acción: "Ver", "Inscribirse", "Descargar", "Conocer"
- Sea específico sobre lo que ocurrirá al hacer clic
- Pruebe que todos los enlaces funcionen correctamente

⛔ **Evite:**
- Imágenes borrosas o de baja calidad
- Textos muy largos que sean difíciles de leer
- Demasiada información en un solo slide
- Enlaces rotos o que no funcionan
- Slides sin un propósito claro

## 5.8. Buenas Prácticas

### ✅ Redacción de Contenido

**Título:**
- Use mayúsculas solo al inicio y en nombres propios
- Sea específico y descriptivo
- Evite clickbait o títulos sensacionalistas
- Longitud ideal: 50-70 caracteres

**Resumen:**
- Responda las preguntas: ¿Qué?, ¿Quién?, ¿Cuándo?, ¿Dónde?
- Use el primer párrafo de la noticia como base
- Sea objetivo y claro
- Evite abreviaturas no estándar

**Contenido:**
- Estructure en párrafos cortos (3-4 líneas)
- Use subtítulos para dividir secciones largas
- Incluya información relevante en los primeros párrafos
- Revise ortografía y gramática antes de publicar
- Use un tono profesional e institucional

### ✅ Imágenes

**Calidad:**
- Resolución mínima: 1200 x 800 píxeles
- Formato JPG para fotografías
- PNG para imágenes con transparencia o gráficos
- Optimice las imágenes antes de subir (use TinyPNG o similar)

**Contenido de la imagen:**
- Relacionada directamente con la noticia
- De preferencia, fotografías propias del evento/situación
- Evite imágenes de stock genéricas
- Si usa imágenes de terceros, asegure tener los derechos

**Composición:**
- Imagen horizontal (landscape)
- Elementos importantes en el centro
- Buena iluminación y enfoque
- Evite texto dentro de la imagen (salvo logos)

### ✅ Publicación

**Frecuencia:**
- Publique al menos 2-3 noticias por mes
- Mantenga una cadencia regular
- Aproveche eventos importantes para generar contenido

**Timing:**
- Publique noticias en horario laboral (9am - 6pm)
- Eventos: publique con al menos 1 semana de anticipación
- Resúmenes de eventos: publique dentro de las 48 horas siguientes

**Revisión:**
- Revise la noticia al menos dos veces antes de publicar
- Verifique que todos los enlaces funcionen
- Confirme fechas, nombres y datos exactos
- Pida a un colega que revise contenido importante

### ✅ Gestión de Noticias Antiguas

**Archivo:**
- Desactive noticias de más de 1 año que ya no sean relevantes
- No las elimine (conserve el histórico)
- Mantenga activas solo noticias de interés actual

**Destacadas:**
- Rote las noticias destacadas mensualmente
- Remueva el estado "Destacado" de noticias antiguas
- Priorice noticias recientes y de alto impacto

### ⛔ Errores Comunes a Evitar

- Títulos en mayúsculas completas (EVITAR ESTO)
- Contenido sin estructura (párrafos muy largos)
- Exceso de negritas o cursivas
- Imágenes de baja calidad o pixeladas
- Información incompleta (especialmente fechas y lugares)
- Errores ortográficos o gramaticales
- Lenguaje informal o coloquial
- Noticias sin categoría asignada

## 7.7. Cambio de Estado

El estado de un colegiado (Activo/Inactivo) es fundamental y se sincroniza con sus habilitaciones.

### Colegiado Activo
- Aparece en el directorio público (si perfil no está oculto)
- Sus habilitaciones están activas
- Puede ejercer profesionalmente

### Colegiado Inactivo
- NO aparece en el directorio público
- Sus habilitaciones se marcan como inactivas
- No puede ejercer hasta ser reactivado

### Cómo cambiar el estado

**Método 1: Desde el perfil**
1. Abra el perfil del colegiado
2. Haga clic en **"Editar Información"**
3. Marque o desmarque "Estado Activo"
4. Guarde los cambios

**Método 2: Al revocar habilitación**
- Al revocar una habilitación, el colegiado pasa automáticamente a INACTIVO

**Método 3: Al reactivar habilitación**
- Al reactivar una habilitación, el colegiado pasa automáticamente a ACTIVO

> ⚠️ **Importante:** El estado del colegiado y sus habilitaciones están sincronizados. Cambiar uno afecta al otro.

### 7.8.2. Sistema de Código QR

Cuando sube un documento de habilitación, el sistema ejecuta un proceso automatizado complejo:

**Proceso automático (el usuario no necesita hacer nada):**

1. **Generación de Código Único**
   - El sistema genera un código único con formato: `HC-{UUID}`
   - Ejemplo: `HC-a7f3e8d9-12c4-4b5a-9f6e-8d7c5b4a3e2f`
   - El código se valida en la base de datos para garantizar unicidad
   - Este código es PERMANENTE y único en todo el sistema

2. **Creación del Código QR**
   - Se genera una imagen QR de 350x350 píxeles
   - El QR contiene la URL de verificación corta: `/v/{codigo}`
   - Label del QR: "CPAP - {Nombre del Colegiado}"
   - Alta corrección de errores (permite leer incluso si está parcialmente dañado)
   - Se guarda en: `storage/public/images/qr/{codigo}.png`

3. **Embebido en el PDF**
   - El sistema toma el PDF original
   - Usa la librería FPDI para manipular el PDF
   - En cada página del documento, agrega en la esquina inferior derecha:
     - El código QR (35mm de ancho)
     - El código de verificación en texto (Arial Bold, 6.5pt)
     - La URL de verificación: /v/{codigo} (Arial, 4.5pt)
     - Marca de agua "Documento Verificable" (color vino CPAP, 8pt)
   - Respeta la orientación y tamaño original de cada página
   - Genera un nuevo PDF con el QR embebido

4. **Almacenamiento**
   - El PDF modificado se guarda en: `storage/public/habilitaciones/`
   - El código QR se guarda en: `storage/public/images/qr/`
   - Se crea un registro en la base de datos con:
     - Código único
     - Ruta del PDF
     - Ruta del QR
     - Relación con el colegiado
     - Usuario admin que lo generó
     - Fecha de generación
     - Estado: ACTIVO

5. **Actualización de Estado**
   - El colegiado pasa automáticamente a estado ACTIVO
   - Si tenía una habilitación anterior, se elimina completamente

# 19. SOLUCIÓN DE PROBLEMAS COMUNES

## 19.1. No Puedo Iniciar Sesión

**Posibles causas:**
- Credenciales incorrectas
- Cuenta bloqueada temporalmente
- Usuario sin permisos

**Acciones:**
1. Restablecer contraseña.
2. Verificar usuario activo.
3. Contactar administrador principal.

## 19.2. Error al Subir Imágenes

**Posibles causas:**
- Formato no permitido
- Tamaño excesivo
- Nombre de archivo inválido

**Acciones:**
1. Convertir a JPG/PNG.
2. Reducir peso.
3. Renombrar sin caracteres especiales.

## 19.3. Error al Subir PDFs

**Posibles causas:**
- PDF dañado
- Tamaño excedido
- Restricciones del navegador

**Acciones:**
1. Reexportar PDF.
2. Reducir tamaño.
3. Probar en otro navegador.

## 19.4. No se Generó el QR

**Posibles causas:**
- Documento no válido
- Proceso interrumpido
- Error temporal del servidor

**Acciones:**
1. Reintentar carga del documento.
2. Verificar integridad PDF.
3. Revisar logs del sistema.

## 19.5. Los Cambios No se Reflejan

**Posibles causas:**
- Caché del navegador
- Estado inactivo del contenido
- Demora de propagación

**Acciones:**
1. Refrescar sin caché (Ctrl+F5).
2. Verificar checkbox activo/publicado.
3. Revalidar desde ventana incógnito.

## 19.6. Problemas con la Búsqueda

**Posibles causas:**
- Término muy específico
- Filtros activos restringiendo resultados

**Acciones:**
1. Limpiar filtros.
2. Probar términos parciales.
