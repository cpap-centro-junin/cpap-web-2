# Objetivos y Alcance del Proyecto

## Objetivo General

Desarrollar e implementar un nuevo sitio web institucional para el Colegio Profesional de Antropólogos del Perú – Región Centro, utilizando tecnologías modernas de desarrollo web y el framework Laravel, con la finalidad de:

- 🎯 Mejorar la difusión de información institucional
- 🎯 Fortalecer la imagen digital de la institución
- 🎯 Brindar una plataforma segura, funcional y escalable
- 🎯 Permitir el crecimiento futuro del sistema

---

## Objetivos Específicos

### 1. Análisis
Analizar la situación actual del sitio web institucional y recopilar los requerimientos necesarios para el desarrollo del nuevo sistema web.

### 2. Diseño
Diseñar la estructura y arquitectura del sitio web, considerando:
- Criterios de usabilidad
- Accesibilidad
- Organización de la información

### 3. Frontend
Desarrollar el frontend del sitio web institucional, asegurando:
- Diseño moderno y responsivo
- Coherencia con la identidad institucional del CPAP – Región Centro

### 4. Backend
Implementar el backend del sistema web utilizando Laravel, garantizando:
- Correcta gestión de datos
- Seguridad del sistema

### 5. Base de Datos
Integrar una base de datos que permita la administración eficiente de la información institucional.

### 6. Pruebas
Realizar pruebas de funcionamiento para asegurar la estabilidad y correcto desempeño del sitio web.

### 7. Implementación
Implementar el sitio web en un servidor de hosting, asegurando su disponibilidad y correcto acceso en internet.

### 8. Administración
Facilitar la administración básica de contenidos del sitio web para el personal autorizado de la institución.

---

## Alcance del Proyecto

### ✅ Incluido en el Proyecto

#### Secciones del Sitio Web
- **Página de inicio**
- **Información institucional:** Reseña, misión, visión, objetivos
- **Consejo directivo**
- **Comunicados y noticias institucionales**
- **Eventos académicos y profesionales**
- **Documentos y normativas institucionales**
- **Sección de contacto**

#### Funcionalidades Técnicas
- Sistema básico de administración de contenidos
- Implementación de medidas de seguridad básicas
- Diseño responsivo (adaptación a diferentes dispositivos)
- Publicación en servidor con dominio propio
- Certificado SSL para navegación segura

---

### ❌ Fuera del Alcance

Lo siguiente **NO** forma parte del proyecto:

- ❌ Desarrollo de aplicaciones móviles
- ❌ Implementación de sistemas de pago en línea
- ❌ Integración de plataformas externas de gestión de colegiaturas
- ❌ Sistemas de trámites administrativos complejos
- ❌ Mantenimiento permanente posterior a las prácticas (salvo coordinaciones adicionales)

---

## Resultados Esperados

Al finalizar el proyecto, se espera obtener:

1. ✅ Sitio web institucional moderno, funcional y accesible
2. ✅ Mejora significativa de la imagen institucional y presencia digital
3. ✅ Plataforma web desarrollada con tecnologías modernas, segura y escalable
4. ✅ Organización clara y eficiente de la información institucional
5. ✅ Sistema web preparado para futuras ampliaciones
6. ✅ Aplicación práctica de conocimientos en un entorno real
